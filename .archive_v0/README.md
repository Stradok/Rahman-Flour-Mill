# Archive - Historical & Deprecated Files

This folder contains files from earlier phases of development that are no longer actively used.

## Contents

- **cloud.md** - Original cloud-based plan (superseded by local-only architecture)
- **prisma_historical/** - Old Prisma schema and migrations (replaced by Drizzle ORM)
- **tmp-spike-encrypted-sqlite.mjs** - Prototype encryption test script

## Why Archived?

The project evolved from a cloud-based system (Neon + Prisma) to a **local-only encrypted desktop application**. These files document the journey but are not part of the current implementation.

See **../docs/** for current architecture and **../CONTRIBUTING.md** for active development.
